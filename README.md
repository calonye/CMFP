# 中医四气：CMFP=Chinese_Medicine_four_properties

- 项目起源：受病因影响，日常饮食受到困扰，查阅资料又非常难理解，且复杂；

- 项目目的：建立平民版低门槛理解方式的四气表格，类别收集普通日常较为常见的饮食方式，鼠标悬浮对应的食材可了解该实在在不同场景下的四气变化；
